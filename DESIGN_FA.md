# KBO 매니저 — FA(자유계약) 설계 (3단계-3)

> 로드맵 3번. 연봉/계약·드래프트 위에 얹는다. **핵심은 경매식 FA 시장** — 여러
> 구단이 한 FA에 오퍼하고, 선수가 돈·출전기회·팀경쟁력을 종합 판단해 선택한다.
> 이게 FM식 서사("노장이 우승 노리고 강팀에 싸게", "젊은 선수가 출전 보고 약팀에")의
> 발생 지점이다.
>
> ※ 이 문서는 **제안**이다. 승인 후 구현.

## 0. 원칙 / 범위 · 이미 있는 것 활용

- **시즌 간 로직**. 오프시즌 체인: `season → 에이징(은퇴) → FA(베테랑 이동) →
  드래프트(잔여 구멍 refill) → 재정(계약·예산·캡·하한)`. 단일 시즌 무영향(회귀 가드).
- **재사용**: `contract_value`/`fair_salary`(입찰가) · `Team.budget`/캡/하한(실탄) ·
  `need_bonus`/`_depth`(출전기회=포지션 뎁스 부족도) · 순위/전력(팀경쟁력) ·
  `service_years`(자격, 연봉단계 훅).
- **이번 범위(핵심만)**: 등급제 A/B/C, **보상금만**(보상선수 생략), 경매식 입찰.
  제외(나중): 보상선수·사인앤트레이드·옵션·multi-year 세부.
- 상수는 `TUNE["fa"]`.

## 1. 자격 & 등급

- **자격**: `service_years >= fa_service_req`(고졸 8/대졸 7 → 우리는 단일 ~8로 근사).
  `service_years`는 연봉단계에서 시즌당 적립(1군 등록 145일≈결장 45경기 이하면 +1,
  아니면 +0.5) — **부상 잦으면 자격이 늦어진다**(부상 시스템 연결). FA로 계약하면
  `fa_eligible_at = service_years + fa_reelig`로 재자격 시점을 미룬다.
- **부트스트랩**(구현 반영): 초기 로스터는 서비스타임이 없으므로 나이−20으로 시딩.
  단 일시 자격 홍수를 막기 위해 재자격 시점을 나이 기반으로 `reelig`년에 스태거
  → 시즌당 자격자 30~50명 수준으로 분산 (실제 KBO 신청자 규모대).
- **등급** (최근 3시즌 연봉 기준, 실제 KBO 근사): FA 신청자를 최근 연봉으로 정렬해
  상위 1/3 = **A**, 중간 1/3 = **B**, 나머지 = **C**. 등급이 보상금을 정한다(§3).

## 2. 선수의 오퍼 종합 판단 — 경매식 선택 ★핵심

한 FA `p`가 받은 오퍼 `o`(구단 `T`, 제시 AAV)의 매력도:

```
appeal(p, o) = w_money(p)·money(o) + w_play(p)·play(p,T) + w_win(p)·win(T)
```

세 점수는 0~1 정규화:
- **money(o)** = `offer_aav(o) / (그 선수가 받은 최고 오퍼 AAV)` — 최고 조건이 1.0.
- **play(p,T)** = `need_frac(T, pos(p))` — 그 구단 해당 포지션 뎁스 부족도(드래프트
  Need 재사용). 약점 포지션일수록 "주전으로 뛴다" → 1.0.
- **win(T)** = 팀경쟁력 = `clamp(0.5 + (중위순위 − 최근순위(T)) / 순위폭, 0, 1)` —
  우승권일수록 1.0.

**가중치 `w`는 선수마다 성향 편향**(FM식 서사의 핵심):
```
base = (money 0.50, play 0.25, win 0.25)
나이 편향:  노장(≥ vet_age 32) → +win, −play (우승 노림)
            젊은 FA(≤ young_age 28) → +play, −money (출전기회 우선)
+ 선수별 노이즈(작은 gauss) 후 clamp≥0, 합=1 정규화 → 성향 다양성
```
- 노장이 우승권 강팀(win 높음)에 **적정가 이하로도** 가는 케이스,
  젊은 선수가 약팀(play 높음)에 출전 보장 받으러 가는 케이스가 창발.
- **잔류**: 원소속팀은 항상 잔류 오퍼(적정가 + `loyalty` appeal 가산)를 낸다.
  외부 최고 매력도가 잔류 매력도를 못 넘으면 **잔류**(대다수가 잔류하는 현실 재현).
- 선택은 최고 appeal(+ 작은 노이즈)로 결정.

## 3. 등급별 보상금 — 이동 브레이크

- **A 300% / B 200% / C 150%** (전년 연봉 대비). **영입팀이 원소속팀에 지급**,
  영입팀 예산에서 차감·원소속팀 revenue 가산.
- 효과: A등급 특급은 보상금 부담이 커 영입팀이 신중 → 이동 유동성 낮음. C등급은
  가벼워 활발. **등급별 이동률 C > B > A**가 자연 발생(§검증).

## 4. 구단(AI) 입찰 로직

각 구단 `T`가 어떤 FA에 얼마를 부를지:

- **대상 선정**: FA 풀에서 `T`의 약점 포지션(`need_frac > need_min`) 중
  `need_frac × asset_war` 상위 → 예산 여유 안에서 타깃 리스트.
- **입찰가(AAV)**:
  ```
  bid_aav = fair_aav(p) × (1 + premium)
  premium = overpay_need · need_frac(T,pos) + overpay_win · contention(T)
  ```
  약점이 절실하거나(need) 우승권 총력(contention)이면 프리미엄 오버페이(장현식
  사례처럼 캡 근처서도 무리 영입). `premium`은 `overpay_cap`으로 상한.
- **총비용 제약**(구현 반영): 시장당 **누적 FA 지출(AAV+보상금) ≤ 예산 × spend_frac**
  (0.40). 예산-페이롤 스케일 차이로 '여유=예산−페이롤'이 음수가 되는 문제를 피하고,
  부자 구단이 더 쓸 수 있는 부익부 축은 유지(드래프트가 상쇄).
- **보상금 합리성**(구현 반영): `보상금 > 계약총가치 × comp_tolerance`면 입찰 포기 —
  가치 5억 계약에 보상금 14억을 내는 비합리 영입 차단. B/C 저연봉 베테랑의 이동이
  보상금에 막히는 실제 현상이 이 규칙에서 창발.
- **영입 인원 제한**: 당해 신청 `N`명 → 팀당 최대 `max(1, N // 10)`. 폭식 방지.
- **로스터 초과 방출**: FA 영입으로 25명 초과 시 최저 가치 선수 방출 (드래프트가
  부족 팀을 refill).
- **하한 미달 팀**: 남는 예산으로 FA 영입해 하한을 채우는 연결(연봉단계 훅 소비).

## 5. 시장 진행 (경매 루프)

```
1) 자격자 중 FA 선언 풀 구성 + 등급 산정.
2) 반복(패스가 무성과일 때까지):
   - 각 구단이 예산·인원 한도 내에서 미계약 타깃에게 최적 입찰가 오퍼.
   - 각 미계약 FA가 (외부 오퍼 + 원소속 잔류 오퍼) 중 최고 appeal 선택.
     · 잔류 선택 → 원팀 재계약, FA 풀에서 제거.
     · 이적 선택 → 영입팀 로스터 편입·원팀 구멍, 보상금 이전, 풀에서 제거.
3) 남은 자격자는 잔류(오퍼 부족). 이후 드래프트가 이적으로 생긴 구멍을 refill.
```
- 여러 구단이 한 대어에 몰리면 그 선수의 money 점수 경쟁 → 프리미엄 높은 팀이
  낙찰(오버페이·승자의 저주 창발). (상승식 경매는 이후 정교화 — 지금은 봉인식
  단발 + 프리미엄.)

## 7. 검증 계획 — `scripts/fa_check.py`

1. **시장 sanity**: 매 오프시즌 이동/잔류 수 — 대다수 잔류 + 소수 대어 이동
   (실제 KBO스러운 규모).
2. **등급별 이동률 C > B > A**: 보상금 브레이크 작동 확인.
3. **FM식 서사 샘플**: '노장이 우승권 강팀에 싼값' · '젊은 FA가 약팀에 출전기회
   보고' 케이스 몇 개 추출(나이·오퍼·appeal 성향 표기).
4. **오버페이/승자의 저주**: 대어에 복수 구단 경쟁 → 낙찰가가 적정가 위로
   가는지(경쟁 입찰 작동). 단 비현실적 폭등은 아닌지(`overpay_cap`).
5. **★ 전력 평준화 영향 (최우선)**: FA는 돈 있는 팀이 대어를 쓸어가는 '부익부'
   위험 — 드래프트(역순=약팀 유리)와 **반대 방향**. 30시즌 순위 순환이 두 힘의
   균형으로 여전히 유지되는지 **4시드+** 재검증. FA가 강팀 독식으로 순환을
   깨면 오버페이/인원한도/보상금을 조여 조정.
6. **회귀 가드**: 단일 시즌 경기 지표 불변.

### 승인받을 핵심 결정
1. **경매식 선택** — 선수가 `w_money·money + w_play·play + w_win·win`로 오퍼 선택,
   가중치는 나이/성향 편향 → FM식 서사 창발
2. **구단 AI 입찰** — `fair_aav × (1 + need·overpay + contention·overpay)`, 보상금
   포함 예산 제약
3. **보상금만** A300/B200/C150% (이동 브레이크), 보상선수는 나중
4. **잔류 우세** — 원팀 잔류 오퍼 + loyalty로 대다수 잔류
5. **★ 순위 순환 재검증** — FA 부익부 vs 드래프트 평준화 균형 (4시드+)
