# KBO 매니저 빠른 시작

## Windows

1. 릴리스 ZIP을 내려받아 압축을 풉니다.
2. 압축을 푼 폴더에서 **`게임 시작.bat`**를 더블클릭합니다.
3. 처음 실행할 때만 전용 환경과 필수 구성요소가 자동 설치됩니다.
4. 설치가 끝나면 브라우저가 자동으로 열립니다.
5. 구단을 선택하고 **새 커리어 시작**을 누릅니다.

게임을 종료할 때는 먼저 화면 오른쪽 위의 **저장**을 누르고, 검은 실행 창에서 `Ctrl+C`를 누르거나 창을 닫습니다.

## macOS·Linux

터미널에서 압축을 푼 폴더로 이동한 뒤 실행합니다.

```bash
chmod +x start_kbo_manager.sh
./start_kbo_manager.sh
```

## 직접 실행

Python 3.10 이상이 설치돼 있다면 다음 명령으로 실행할 수 있습니다.

```bash
python -m pip install -r requirements.txt
python run_game.py
```

기본 주소는 `http://127.0.0.1:8000`입니다. 8000번 포트가 사용 중이면 실행기가 8010번까지 빈 포트를 자동으로 찾습니다.

## 첫 커리어 진행 순서

1. 구단 선택
2. 대시보드에서 구단주 목표 확인
3. `하루`, `시리즈`, `한 달` 또는 `시즌 끝까지`로 진행
4. 내 팀 경기는 `직접 운영` 또는 일정 화면에서 관전
5. 시즌 중 구단주 이벤트에 응답
6. 오프시즌에서 트레이드·FA·보상선수·드래프트 진행
7. 시즌 종료 평가와 감독 커리어 기록 확인

화면 상단의 **도움말** 버튼에서 같은 내용을 다시 볼 수 있습니다.

## 저장 파일

저장 파일은 게임 폴더 아래 `saves/save.pkl`에 생성됩니다. 새 버전으로 옮길 때는 `saves` 폴더를 함께 복사하세요.

## 문제 해결

### Python을 찾을 수 없다고 나오는 경우

Python 설치 시 **Add python.exe to PATH**를 체크하지 않은 경우입니다. Python 설치 프로그램을 다시 실행해 PATH 옵션을 활성화하세요.

### 설치 중 인터넷 오류가 나는 경우

최초 실행은 `fastapi`, `uvicorn`, `httpx` 설치를 위해 인터넷 연결이 필요합니다. 설치가 끝난 뒤에는 오프라인에서도 실행할 수 있습니다.

### 브라우저가 자동으로 열리지 않는 경우

실행 창에 표시된 주소를 Chrome 또는 Edge 주소창에 직접 입력하세요.

### Address already in use

실행기는 보통 다음 빈 포트를 자동 선택합니다. 직접 실행 중이라면 다음처럼 다른 포트를 지정하세요.

```bash
python run_game.py --port 8001
```

### 저장된 게임을 불러올 수 없는 경우

- `saves/save.pkl` 파일이 있는지 확인합니다.
- 다른 버전의 폴더로 옮겼다면 `saves` 폴더 전체를 복사합니다.
- 저장 파일은 신뢰할 수 있는 본인 파일만 사용하세요.

### 화면이 비어 있거나 오류가 반복되는 경우

1. 브라우저에서 `Ctrl+F5`로 새로고침합니다.
2. 실행 창을 종료하고 `게임 시작.bat`를 다시 실행합니다.
3. 그래도 해결되지 않으면 실행 창의 오류 메시지와 게임 버전을 함께 기록합니다.
