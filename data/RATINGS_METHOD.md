# 선수 능력치 산정 방법 (players.json)

능력치는 임의 창작이 아니라 **각 선수의 최근 시즌(2024~2025) 실제 성적을 앵커 테이블로 역산**해 부여했다.
출처는 모델이 학습한 KBO 공개 기록(타율/홈런/도루/볼넷율/삼진율/ERA/K9/BB9/이닝/세이브 등)이며,
스크래핑한 원본 데이터셋이 아니므로 **근사치**다. 각 선수의 `basis` 필드에 산정 근거를,
확신이 낮은 선수(백업/신규 외국인/최근 기록 불확실)는 `"est": true`로 표기했다.

기준: **리그 평균 = 50** (엔진의 log5 수식이 50을 리그 평균으로 취급한다).
실제 로스터는 주전 위주라 PA 가중 평균이 50을 넘는데, 이는 캘리브레이션의 앵커 상수로 보정한다.

## 타자 매핑 앵커

| 능력치 | 근거 스탯 | 앵커 |
|---|---|---|
| 컨택 (con) | 타율 + 삼진율 | .330+/K%<12 → 85~95 · .300대 → 75~84 · .280대 → 65~74 · .260대 → 55~64 · .240대 → 45~54 · 그 이하 → 35~44 |
| 파워 (pow) | 홈런(600PA 환산) + ISO | 35+ → 88~95 · 25~34 → 78~87 · 15~24 → 65~77 · 8~14 → 52~64 · 그 이하 → 35~51 |
| 선구안 (eye) | BB% (≈OBP−AVG) | 13%+ → 85+ · 10~12% → 72~84 · 8~9% → 58~71 · 6~7% → 45~57 · <6% → 30~44 |
| 주루 (spd) | 도루 + 주루 평판 | 30+ SB → 88+ · 15~29 → 72~87 · 5~14 → 55~71 · 둔중/포수 → 28~50 |
| 수비 (fld) | 수비 평판·골든글러브·포지션 소화력 | **정성 평가** (±10 오차 가정) |
| 송구 (arm) | 송구 평판 (포수는 도루저지율) | **정성 평가** (±10 오차 가정) |

| 능력치 | 근거 스탯 | 앵커 |
|---|---|---|
| 구속 (vel) | 평균 구속 | 150km/h+ → 85+ · 147~149 → 75~84 · 144~146 → 62~74 · 141~143 → 50~61 · 이하 → 35~49 (잠수함/기교파는 실효 위력 감안 소폭 보정) |
| 제구 (ctl) | BB/9 | <1.8 → 85+ · 2.0~2.5 → 72~84 · 2.6~3.3 → 58~71 · 3.4~4.2 → 45~57 · >4.5 → 30~44 |
| 구위 (stf) | K/9 + 피안타·FIP | K9 9+ → 82+ · K9 8대 → 72~81 · 7대 → 62~71 · 6대 → 52~61 · 5대 이하 → 40~51 |
| 스태미나 (sta) | 시즌 이닝(선발) | 170IP+ → 85+ · 150대 → 75~84 · 130대 → 65~74 · 100대 → 55~64 · 불펜 → 28~45 (멀티이닝형 +5~10) |
| 변화구 (brk) | 결정구 위력 평판 + 헛스윙/땅볼 성향 | 정성 평가 + K% 보조 |

## 공통 규칙

- **나이**: 2026 시즌 기준.
- **연봉/계약기간**: 실제 공개 계약(FA 총액/외국인 연봉)을 연 단위로 근사. 비FA는 연차·성적 기반 추정. 외국인은 전원 1년.
- **좌/우 (bats/throws)**: 실제 선수의 타석/투구 방향. 불확실하면 우로 두고 `est` 표기.
- **신규 외국인 선수**: KBO 기록이 없으므로 MLB/마이너 성적을 KBO 환산해 추정 (`est`).
- **백업 뎁스(팀당 3~5명)**: 1군 출장 기록이 적어 대부분 `est`. 로스터 25명(야수 14+투수 11) 구성을 위한 실존 선수다.
- 2025 시즌 종료 후 이적/은퇴 반영이 일부 부정확할 수 있다 (팬게임 근사치).
