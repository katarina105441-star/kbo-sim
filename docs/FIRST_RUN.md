# KBO 매니저 첫 실행 가이드

이 문서는 코딩을 몰라도 게임을 켜고 첫 시즌을 진행할 수 있도록 정리한 안내입니다.

## 1. 설치

### Windows 기준

1. Python을 설치합니다.
   - python.org에서 최신 Python 3을 설치합니다.
   - 설치 첫 화면에서 **Add python.exe to PATH**를 반드시 체크합니다.
2. GitHub 저장소에서 초록색 **Code** 버튼을 누른 뒤 **Download ZIP**을 선택합니다.
3. ZIP을 압축 해제합니다.
4. 압축을 푼 폴더에서 주소창에 `cmd`를 입력해 명령창을 엽니다.
5. 최초 1회만 아래 명령을 실행합니다.

```bash
python -m pip install -r requirements.txt
```

## 2. 실행

압축을 푼 폴더에서 아래 명령을 실행합니다.

```bash
python -m uvicorn web.backend.main:app --port 8000
```

명령창에 `Uvicorn running on http://127.0.0.1:8000` 비슷한 문구가 보이면 성공입니다.
그 창은 닫지 말고 그대로 둡니다.

브라우저 주소창에 아래 주소를 입력합니다.

```text
http://localhost:8000
```

## 3. 첫 시즌 진행 순서

1. 첫 화면에서 운영할 구단을 선택합니다.
2. 시드는 비워도 됩니다. 같은 결과를 반복 검증하고 싶을 때만 숫자를 넣습니다.
3. **팀으로 시작** 버튼을 누릅니다.
4. 대시보드에서 다음 중 하나를 선택합니다.
   - **직접 운영**: 오늘 경기를 직접 진행합니다.
   - **하루**: 하루치 경기를 자동 진행합니다.
   - **시리즈**: 3일치 경기를 자동 진행합니다.
   - **한 달**: 한 달치 경기를 자동 진행합니다.
   - **시즌 끝까지**: 다음 이벤트 또는 시즌 종료까지 진행합니다.
5. 중간에 구단주 이벤트, 트레이드, FA, 보상선수, 드래프트가 나오면 먼저 처리해야 계속 진행됩니다.
6. 상단 **저장** 버튼을 자주 누릅니다.

## 4. 탭별 역할

| 탭 | 역할 |
| --- | --- |
| 대시보드 | 시즌 목표, 구단주 신뢰도, 팬/언론 반응, 시즌 진행 |
| 순위표 | 현재 리그 순위 확인 |
| 일정·결과 | 경기 결과, 박스스코어, 관전 |
| 로스터 | 전체 선수단과 선수 상세 확인 |
| 라인업 관리 | 타순, 수비 위치, 선발 로테이션, 불펜 보직 조정 |
| 2군·육성 | 1군/2군 이동, 육성 방향 설정 |
| 오프시즌 | 트레이드, FA, 보상선수, 드래프트 진행 |

## 5. 진행이 막히는 정상 상황

다음 상황은 오류가 아니라 게임 규칙입니다.

- **구단주 이벤트에 먼저 응답해야 합니다**: 대시보드의 선택지를 고릅니다.
- **트레이드/FA/보상선수/드래프트를 먼저 처리해야 합니다**: 오프시즌 탭으로 이동합니다.
- **재취업할 구단을 먼저 선택해야 합니다**: 해임 후 대시보드의 제안 카드에서 새 구단을 고릅니다.
- **은퇴한 감독의 커리어는 더 진행할 수 없습니다**: 커리어가 종료된 상태입니다.

## 6. 종료와 이어하기

1. 게임 화면 상단의 **저장**을 누릅니다.
2. 명령창에서 `Ctrl+C`를 누릅니다.
3. 다음 실행 때는 같은 폴더에서 다시 아래 명령을 실행합니다.

```bash
python -m uvicorn web.backend.main:app --port 8000
```

4. 브라우저에서 `localhost:8000` 접속 후 **저장된 게임 불러오기**를 누릅니다.

## 7. 자주 나는 문제

| 증상 | 해결 |
| --- | --- |
| `'python'은(는) 인식할 수 없습니다` | Python 설치 시 PATH 체크가 빠진 것입니다. Python 설치 파일을 다시 실행해 PATH를 체크합니다. |
| `'pip'은(는) 인식할 수 없습니다` | `python -m pip install -r requirements.txt`로 실행합니다. |
| `Address already in use` | 이미 서버가 켜져 있습니다. 기존 명령창을 닫거나 `--port 8001`로 실행하고 `localhost:8001`로 접속합니다. |
| 브라우저에 게임이 없다고 나옴 | 정상입니다. 구단을 선택해 새 게임을 시작하거나 저장 파일을 불러오면 됩니다. |
