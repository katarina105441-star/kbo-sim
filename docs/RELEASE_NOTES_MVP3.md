# KBO 매니저 MVP-3 릴리스 노트

## 요약

MVP-3는 단순 시즌 시뮬레이터에서 **구단 운영·오프시즌 개입·감독 커리어 게임**으로 확장한 릴리스입니다.

사용자는 구단을 선택해 시즌을 진행하고, 경기 직접 운영, 라인업 관리, 2군 육성, 트레이드, FA, 보상선수, 드래프트, 구단주 목표, 해임·재취업·은퇴까지 경험할 수 있습니다.

## 주요 기능

### 경기·시즌 진행

- 하루, 시리즈, 한 달, 시즌 끝까지 자동 진행
- 오늘 경기 직접 운영
- 투수 교체, 대타, 대주자, 대수비
- 경기 관전과 박스스코어
- 스포일러 방지형 결과 확인

### 구단 운영

- 라인업과 투수 보직 편집
- 2군/1군 이동
- 육성 방향 설정
- 구단별 운영 성향
- 구단별 드래프트·FA·트레이드 판단 차이

### 오프시즌

- 직접 트레이드 협상
- FA 직접 입찰
- FA 보상선수 보호/지명
- 신인 드래프트 직접 지명
- 오프시즌 저장·이어하기

### 프런트 평가와 커리어

- 시즌 목표
- 구단주 신뢰도
- 해임 위험
- 구단주 이벤트
- 목표 달성 보상
- 감독 업적
- 실제 해임
- 재취업 제안
- 구단 이동
- 팬 지지율과 언론 압박
- 10시즌 이후 자발 은퇴
- 30시즌 자동 은퇴
- 레거시 점수와 명예의 전당

## 저장 파일 호환

기존 저장 파일은 자동 마이그레이션됩니다.
누락된 필드는 기본값으로 복원됩니다.

다만 MVP-3 이전 저장 파일은 새 시스템의 과거 이력을 갖고 있지 않으므로, 불러온 이후부터 커리어·업적·프런트 평가가 누적됩니다.

## 검증 상태

- Python 전체 테스트 통과
- FastAPI import 통과
- 프론트엔드 빌드 통과
- 4시드 × 30시즌 장기 감독 커리어 밸런스 검증 통과
- 브라우저 실행용 정적 빌드본 포함

## 첫 실행

```bash
python -m pip install -r requirements.txt
python -m uvicorn web.backend.main:app --port 8000
```

브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:8000
```

자세한 안내는 `docs/FIRST_RUN.md`를 확인하십시오.

## 알려진 이슈

- 장기 모드에서 일부 리그 지표 드리프트가 남아 있습니다. 단일 시즌 밸런스는 유지됩니다.
- 브라우저 UI는 로컬 실행 기준입니다. 별도 호스팅 배포 자동화는 다음 단계에서 다룹니다.
- 프론트엔드 개발 의존성은 Vite 5 계열입니다. 보안 감사에서 개발 도구 의존성 경고가 나올 수 있으나, 릴리스 실행본은 정적 빌드로 제공됩니다.
